# Weather-App
A responsive and user-friendly Weather Application built using HTML, CSS, and JavaScript, integrated with a real-time Weather API to fetch live weather data for any city worldwide.

# Features

Search weather by city name

eal-time weather data using external API

Displays temperature, humidity, wind speed, and weather conditions

Responsive and clean UI design

Fast API fetching using fetch() / async-await

Error handling for invalid city names

# Tech Stack

HTML5

CSS3 (Flexbox/Grid)

JavaScript (ES6+)

REST API Integration

# What I Learned

Handling API requests and JSON responses

Working with asynchronous JavaScript

DOM manipulation and dynamic UI updates

Error handling and input validation
